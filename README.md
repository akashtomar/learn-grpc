# learn-grpc
repo to try and learn grpc

## python command
```
python -m grpc_tools.protoc -I../../protos --python_out=. --grpc_python_out=. ../../protos/helloworld.proto
```

## NodeJS command
```
npm i grpc //install grpc package
```
```
npm i @grpc/proto-loader // install dynamic proto reader
```
